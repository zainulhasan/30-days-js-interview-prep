How to use this course

  1. Read the goal and concept for the day.
  2. Watch the animation — press Step to go one move at a time, or Play to run it. The narration line tells you exactly what's happening in plain English.
  3. Code along — read the commented code and the line-by-line explanation.
  4. Solve the 3 practice problems yourself before peeking at hints or solutions.
  5. Quiz yourself with the 5 quick questions, then mark the day complete.
